gitlab仓库yml配置文件搭建

发表于 2021-08-01 更新于 2025-06-28 分类于 gitlab

之前写篇 gitlab 仓库搭建步骤教程，最近升级的时候发现命令实在是太长了，于是决定使用 docker-compose.yml 文件进行配置搭建。

关于上一篇 gitlab 仓库搭建步骤教程可以点击这里

  1. image 为使用的镜像名称
  2. container_name 为容器名
  3. restart 是否自动重启
  4. hostname 主机名，最好是实际的访问地址
  5. GITLAB_OMNIBUS_CONFIG 的配置参数，为 gitlab.rb 参数
  6. ports 端口映射列表
  7. volumes 目录映射列表
  8. logging 日志记录

更新gitlab版本

运行命令 docker-compose pull && docker-compose up -d

下载 docker-compose.yml

gitlab ci git
